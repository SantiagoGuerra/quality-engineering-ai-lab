# QA workflow

Checkout con `gh pr checkout <PR>` o `qa:worktree:create`. Instala, prepara, levanta, elige seed, smoke. Explora reglas/negativos/roles/extremos/red, revisa Mailpit, prueba teclado y AT. Clasifica cualquier fallo antes de Jira. Genera `qa:report`, comparte enlaces pequeños y apaga. No actualiza baseline ni cuarentena sin owners.
